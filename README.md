# Machine Learning Internship Projects

This repository contains all internship tasks completed as part of the Machine Learning Internship Program.

---

## Tasks Included

### Task 1 - Decision Tree Implementation
Built and visualized a Decision Tree model using Scikit-learn to predict restaurant price ranges.

### Task 2 - Sentiment Analysis with NLP
Performed sentiment analysis using TF-IDF Vectorization and Logistic Regression.

### Task 3 - Image Classification
Built an image classification model using machine learning techniques.

### Task 4 - Recommendation System
Developed a restaurant recommendation system using cosine similarity.

- Matplotlib
